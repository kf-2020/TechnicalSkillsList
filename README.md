* オンライン診療予約、ビデオチャットでの診察、決済までをワンストップで完結するオンライン診療システム
	* エンドユーザー様は医療機関スタッフおよび医療機関利用者(患者)
	* 医療機関スタッフ側はブラウザベースのWebアプリ、医療機関利用者(患者)側はAndroid/iOSネイティブアプリをそれぞれ利用想定。
